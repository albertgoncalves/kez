# kez

Bare-bones [OpenGL](https://www.opengl.org//) sprite renderer.

<div align="center"><img src="cover.png"></div>

Needed things
---
*   [Nix](https://nixos.org/download.html)

Quick start
---
```
$ nix-shell
[nix-shell:path/to/kez]$ ./scripts/run.sh
```
